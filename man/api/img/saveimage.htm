主页-＞库函数目录-＞图像处理相关函数-＞saveimage

功能：
这个函数用于将图像保存到文件中。目前只能保存成BMP或者PNG格式。

声明：

int saveimage(
    PIMAGE  pimg,
    LPCSTR  filename,
    image_type_e type = image_type_auto_detect
);

int saveimage(
    PIMAGE  pimg,
    LPCWSTR  filename,
    image_type_e type = image_type_auto_detect
);



参数：
pimg
要保存的图像

filename
保存的图像文件名

type
使用何种图像格式保存文件



image_type_auto_detect根据文件后缀名判断图像格式


image_type_pngPNG格式


image_type_bmpBMP格式




返回值：
成功返回grOk，否则返回非0，若pimg传入错误，会引发运行时异常。
